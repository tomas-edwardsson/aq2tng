AQ2:TNG Online Config Generator

How to use this page:

With TNG we changed some of the behaviour regarding configuration files. You normally need two kinds of configs: the global config and another config for each mode you want to add.

If you don't need different configs, you can still use a single config - just fill out both forms and download the created files. Then simply merge them into one config file and everything is fine.
